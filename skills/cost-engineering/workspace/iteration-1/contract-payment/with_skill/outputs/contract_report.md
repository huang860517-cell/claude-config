## 二、支付条款逐项解析

### 2.1 工程预付款

| 条款项 | 内容 |
|--------|------|
| 支付比例 | 签约合同价的 **15%** |
| 预付款金额 | **8,790,000.00 元** |
| 支付条件 | 承包人提交履约担保 + 预付款担保后 14 天内 |
| 扣回方式 | 从第 1 次进度款起，分 **4 次等额扣回** |
| 每次扣回额 | 2,197,500.00 元 |

### 2.2 工程进度款

| 条款项 | 内容 |
|--------|------|
| 申报周期 | **每月** 25 日前提交已完工程量报告 |
| 审核时限 | 发包人收到报告后 **14 天**内完成审核 |
| 支付比例 | 已完工程量的 **80%** |
| 安全文明施工费 | **随进度款同期支付** |

### 2.3 竣工结算款

| 条款项 | 内容 |
|--------|------|
| 结算资料提交 | 竣工验收合格后 **28 天**内 |
| 结算审核时限 | 发包人收到资料后 **60 天**内 |
| 支付时间 | 审核完成后 **14 天**内 |
| 支付至比例 | 结算价的 **97%** |
| 结算款金额（估算） | 结算价 × 97% − 已付进度款 |

### 2.4 质量保证金

| 条款项 | 内容 |
|--------|------|
| 质保金比例 | 结算价的 **3%** |
| 质保金金额（估算） | 约 **1,758,000.00 元** |
| 缺陷责任期 | **24 个月** |
| 返还条件 | 缺陷责任期满后 **14 天**内 |

### 2.5 变更与签证

| 条款项 | 内容 |
|--------|------|
| 设计变更 | 由设计单位出具变更图纸，发包人确认后实施 |
| 现场签证 | 发生后 **7 天**内办理，逾期不予认可 |
| 变更计价 | 有相同项用相同单价；有类似项参照调整；无相同无类似由承包人报价经发包人确认 |

### 2.6 材料调差

| 条款项 | 内容 |
|--------|------|
| 可调材料 | 钢筋、商品混凝土、水泥 |
| 调差基准期 | 投标截止日前 28 天信息价 |
| 风险幅度 | **±5%**（超出部分调整） |
| 调差周期 | 按月调整，进度款同期支付 |
